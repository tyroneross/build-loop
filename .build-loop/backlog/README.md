# Backlog — .build-loop/backlog/

Host-neutral, filesystem-first deferred-work tracker.

- `items/<ID>.md` is canonical truth.
- `INDEX.md` is generated by `python3 scripts/backlog.py sync --repo .`.
- `archive/<ID>.md` retains done or dropped work.

On an `INDEX.md` conflict, keep both item files and regenerate the index.
