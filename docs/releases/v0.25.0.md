# v0.25.0 Release Notes

## Memory Install And Retrieval

- SQLite is now the default local semantic-facts backend. NavGator lesson sync writes to `<memory-root>/indexes/semantic_facts.sqlite` without requiring Postgres credentials, `psycopg`, or an embedding backend.
- Postgres semantic-facts writes are now an explicit mirror path via `scripts/sync_navgator_lessons.py --postgres-mirror`.
- `memory_facade.recall()` reads the local SQLite semantic index first. `--skip-postgres` still prevents Postgres imports and connection attempts while preserving local semantic recall.
- Fresh installs now bootstrap an `indexes/` lane through `scripts/install_memory.py`.

## Package Surface

- The published package file list now includes the memory installer, memory templates, the graph adapter, the SQLite semantic index, NavGator lesson sync, and optional Postgres mirror helpers.
- The install-verification workflow now checks those package files, runs a fresh memory bootstrap, and exercises `blm context --mode expand` from an installed package.
